# Poorav Desai - Portfolio

My personal portfolio website showcasing my work as a Software Engineer specializing in Cloud & AI Systems. Built with modern web technologies to deliver a fast, responsive, and visually engaging experience.

**Live Site:** [Coming Soon]

## Tech Stack

- **Framework**: Next.js 14+ with App Router
- **Language**: TypeScript
- **Styling**: SCSS, TailwindCSS, CSS Modules
- **Animations**: Framer Motion
- **Icons**: FontAwesome
- **Error Tracking**: Sentry
- **Deployment**: Vercel

## Features

- Fully responsive design optimized for all device sizes
- Smooth animations and interactive UI components
- Project showcase with detailed case studies
- Integrated PDF resume viewer
- Professional sections for experience, skills, and leadership
- Dark theme with custom color scheme
- SEO optimized with sitemap and robots.txt

## Local Development

To run this project locally:

```bash
# Clone the repository
git clone https://github.com/pooravdesai/my-portfolio-nextjs.git

# Navigate to project directory
cd my-portfolio-nextjs

# Install dependencies
npm install
# or
bun install

# Start development server
npm run dev
# or
bun dev
```

Open [http://localhost:4000](http://localhost:4000) in your browser to view the site.

## Deployment

This project is optimized for deployment on [Vercel](https://vercel.com). Simply connect your GitHub repository to Vercel for automatic deployments on every push.
